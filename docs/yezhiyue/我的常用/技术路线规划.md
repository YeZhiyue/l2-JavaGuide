# 前言

<font color=#999AAA > 学习了Java快要 1 年了，终于发现在技术的学习上需要分为两种路线来学习。一是**理论**学习，这个学习关系到你长远的发展和面试，但是比较枯燥无味；二是学习**实战**，这个是比较有趣的地方，因为通过这个可以让你有所成就感，让你感觉到自己知道这个技术可以干嘛，怎么用，但是如果仅仅只有实战但是没有理论基础，就很难走远。两者的学习方式是不一样的，资料的准备也是不一样的。
